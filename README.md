# 💳 Credit Card Generator

A modern Credit Card Generator built with React.js and Tailwind CSS.

## ✨ Features

- Premium UI Design
- Eligibility Validation
- Dynamic Credit Card Preview
- Responsive Layout
- React Hooks
- Tailwind CSS

## 🛠 Tech Stack

- React.js
- Tailwind CSS
- JavaScript
- Vite

## 🚀 Run Locally

```bash
npm install
npm run dev
```
